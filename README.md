# 📊 Quantova

**Quantova** is an intuitive web application for financial management and expense tracking. Designed with a focus on simplicity and user experience, it allows users to record income and expenses, visualize budget distribution through charts, and maintain an organized transaction history.

🔗 **Live Demo:** [Quantova on GitHub Pages](https://samuelmartinsalmeida99-ops.github.io/Quantova/)

---

## ✨ Key Features

* **Central Dashboard:** A quick overview summarizing Expenses, Income, Current Balance, Total Categories, and the Largest Expense.
* **Visual Analysis:** Interactive charts to monitor the distribution of expenses and income by category.
* **Transaction History:** A detailed log of all financial movements, organized and easy to browse.
* **Record Management:** Quick-access floating button to easily add new income or expenses.
* **Customization & Settings:** 
  * Change display currency (Euro, US Dollar, British Pound, Brazilian Real).
  * Clear database option (includes a safety warning to prevent accidental data loss).
